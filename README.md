# AI Customer Support Assistant

A Retrieval-Augmented Generation (RAG) based customer support assistant built using LangChain, FAISS, and Sentence Transformers.

## Features

* Semantic search using vector embeddings
* FAISS vector database
* Document chunking and retrieval
* LangChain integration
* Knowledge base driven responses
* Ready for Llama 3 / OpenAI integration
* Streamlit UI (coming next)

## Tech Stack

* Python
* LangChain
* FAISS
* Sentence Transformers
* Hugging Face Embeddings
* Streamlit
* Ollama (planned)
* Llama 3 (planned)
